### Components & Fields:
- **Email Input**: Regular text input, email validation
- **Password Input**: Secure password field
- **Sign In Button**: POST to `/api/auth/login`, error handling with toast
- **Create Account Button**: Opens signup modal with fields
- **Signup Form Modal**:
  - Name (text)
  - Email (email)
  - Password (password, min 8 chars)
  - Confirm Password (match validation)
  - Phone (10 digits)
  - Address (textarea)
  - Submit button

### Colors:
- Primary Green: `#16a34a`
- Primary Blue: `#3b82f6`
- Text Dark: `#111827`
- Text Gray: `#6b7280`
- Background: `#ffffff` (card), `#f9fafb` (page)
- Borders: `#d1d5db` (light gray)

### States:
- **Loading**: Spinner icon, button disabled
- **Error**: Red toast notification with error message
- **Success**: Green toast, auto-redirect to `/home`
- **Validation Error**: Red border on invalid fields, error text below

### Typography:
```
Headings: 600-700 weight, 16-32px size
Body: 400 weight, 13-16px size

Labels: 600 weight, 13-14px size
Monospace: Product IDs, prices (fixed width)

### Spacing:
```
8px:   Tight (button gaps, small padding)
12px:  Default (card padding, item gaps)
16px:  Medium (section padding, list gaps)
20px:  Large (page padding, major sections)
24px:  Extra Large (section separation)
```

### Border Radius:
```
4px:   Buttons, badges, small elements
6px:   Input fields, small containers
8px:   Cards, modals, medium containers
12px:  Large cards, hero sections
```

```
